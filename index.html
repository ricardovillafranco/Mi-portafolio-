Projects

Vrim Connect Web Page

 Descubre un proyecto destacado que demuestra mi experiencia en desarrollo web. Con WordPress, Elementor y el tema Astra, creé un sitio web dinámico y funcional. Utilicé el poderoso plugin ACF (Advanced Custom Fields) para una personalización óptima. Desde la impactante página de inicio hasta las secciones especializadas como 'Beneficios' y 'Área Médica', cada elemento refleja mi dedicación a la excelencia en diseño y usabilidad.

Koi ecommerce

 Descubre uno de mis proyectos destacados como desarrollador web, donde creé un e-commerce de primera categoría utilizando WordPress, Elementor Pro y el tema Astra. Implementé herramientas avanzadas como CrocoBlock Smart Filters, iVory Search y JoinChat para una experiencia de usuario excepcional. Gracias a mi experiencia en WooCommerce y la implementación de Advanced Custom Fields (ACF), garantizo una experiencia de compra sin igual. ¡Explora mi portafolio y descubre cómo puedo llevar tu próximo proyecto web al siguiente nivel!

Pokemon Game

 ¡Explora mi proyecto personal donde fusioné mi pasión por Pokémon con mi habilidad en el desarrollo web! Este juego interactivo, creado con HTML, CSS y JavaScript, te sumerge en emocionantes combates al estilo 'piedra, papel y tijera' con tus Pokémon favoritos. Adoptando una sólida programación orientada a objetos, cada Pokémon cuenta con habilidades únicas y ataques especiales. Utilicé librerías como JSConfetti para agregar efectos visuales, brindando una experiencia visualmente impactante al ganar un combate. Con una interfaz intuitiva y gráficos cautivadores, este proyecto no solo muestra mi dominio en JavaScript, HTML y CSS, sino también mi capacidad para crear juegos interactivos y entretenidos. ¡Desafía a tus amigos y demuestra quién es el verdadero maestro Pokémon!
